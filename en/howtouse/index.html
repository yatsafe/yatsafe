How to use

YATSAFE safety kit (Small)

You can easily learn how to use the YATSAFE safety kit through video and user manuals.

Information on components

  • All-in-one anchor

  • 1. Anchor insertion

    Insert the anchor into the hole.

  • 2. Anchor combined

    Push the connect bar to combine the anchor.

  • 3. Handle manipulation

    Bend the handle and tighten the anchor as much as possible.

  • 4. Close-up

    Attach the anchor and pad perfectly.

YATSAFE safety kit (large)

You can easily learn how to use the YATSAFE safety kit (large) through videos and instructions.

Information on components

  • Silicone pad
  • Anchor
  • Pad cylinder
  • Connect bar
  • Clamp handle
  • Clamp

  • 1. Insert the anchor

    Insert an umbrella anchor into the perforated area.

  • 2. Anchor combined

    Push the connect bar to combine the anchor.

  • 3. Insert the pad

    Insert the pad and push it all the way in.

  • 4. Pad cylinder

    Insert the clamp handle into the pad cylinder.

  • 5. Conclusion

    Push the clamp handle all the way to bind it.

  • 6. Close-up

    Close the anchor and pad perfectly.
